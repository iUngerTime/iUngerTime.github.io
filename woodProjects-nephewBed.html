Nephew's First Bed

Description

I had the absolute pleasure of spending a weekend with my father designing and building a bed for my growing nephew. With a few projects under my belt by this point, I was able to design quicker and build faster.

Simplicity of design and using the proper tools made this bed really easy to build and cheap. Using a router, we were able to mimic several board in the headboard, while using thin plywood.
